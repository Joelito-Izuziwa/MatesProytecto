Límites

Hablamos de límite cuando se quiere describir procesos de comportamiento tendencial entre magnitudes variables.

Por ejemplo: la determinación de la velocidad de un cuerpo móvil en un tiempo dado, el grado de inclinación de una curva en un punto específico, la rapidez de crecimiento poblacional de una ciudad en un año, y la determinación de áreas bajo la curva definida sobre un intervalo cerrado.

 

Tipos de límites:

Si el límite de una función cuando ( x ) se aproxima a un punto ( a ) es un valor finito ( L ), se dice que el límite existe.

Si el valor de la función crece sin límite cuando ( x ) se acerca a un punto, decimos que el límite es infinito.

Describe el comportamiento de una función cuando ( x ) se aleja hacia valores extremadamente grandes o pequeños.

Formas como ( 0/0 ) o ( ∞/∞ ) requieren el uso de las reglas de L'Hopital para ser resueltas.
